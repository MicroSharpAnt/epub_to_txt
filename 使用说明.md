# EPUB 转 TXT 使用说明

这是一款本地网页工具，用于把 `.epub` 电子书转换成 UTF-8 编码的 `.txt` 文本文件。程序在你的电脑本地运行，上传的 EPUB 文件不会发送到外部服务器。

## 适用场景

- 把 EPUB 小说、文档或资料转换成纯文本。
- 在浏览器中预览转换结果。
- 下载转换后的 `.txt` 文件，便于后续复制、朗读、检索或整理。

## 环境要求

- Python 3.10 或更高版本。
- macOS、Linux 或 Windows 均可运行。
- 一个现代浏览器，例如 Chrome、Edge、Safari 或 Firefox。

## 首次安装

在项目目录中执行：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

如果你使用的是 Windows PowerShell，激活虚拟环境的命令通常是：

```powershell
.venv\Scripts\Activate.ps1
```

## 启动程序

在项目目录中执行：

```bash
uvicorn app.main:app --host 127.0.0.1 --port 8000
```

启动成功后，终端会显示类似下面的地址：

```text
http://127.0.0.1:8000
```

用浏览器打开这个地址即可进入网页界面。

## 网页操作步骤

1. 点击“选择 EPUB 文件”。
2. 选择一个后缀名为 `.epub` 的电子书文件。
3. 点击“转换”。
4. 等待页面显示转换后的文本预览。
5. 确认内容无误后，点击“下载 TXT”。
6. 浏览器会下载一个同名的 `.txt` 文件。

## 转换规则

- 按 EPUB 内部的 spine 阅读顺序提取章节。
- 自动去除 HTML 标签、脚本、样式和目录导航内容。
- 自动解码常见 HTML 实体，例如 `&amp;`。
- 将段落、标题等块级内容转换为文本段落。
- 合并多余空白，保留适合阅读的段落空行。
- 输出文件使用 UTF-8 编码。

## 常见问题

### 页面提示“请选择 .epub 文件”

请选择后缀名为 `.epub` 的文件。普通 `.txt`、`.pdf`、`.docx` 文件不能直接转换。

### 页面提示“上传的文件不是有效的 EPUB”

这个文件可能不是标准 EPUB，或者文件已经损坏。可以尝试重新下载电子书后再转换。

### 转换结果为空

部分 EPUB 是图片扫描版，正文实际存储在图片中。本工具不包含 OCR 功能，因此无法从图片中识别文字。

### 中文文件名下载异常

程序会使用安全的下载响应头处理中文文件名。大多数现代浏览器会保留中文文件名；如果浏览器不支持，会使用 `download.txt` 作为备用文件名。

### 端口 8000 被占用

可以换一个端口启动，例如：

```bash
uvicorn app.main:app --host 127.0.0.1 --port 8010
```

然后打开：

```text
http://127.0.0.1:8010
```

## 停止程序

回到运行 `uvicorn` 的终端窗口，按：

```text
Ctrl+C
```

即可停止本地服务。

## 运行测试

如果你修改了程序，可以执行：

```bash
pytest -v
```

看到全部测试通过后，再使用或提交修改。
